# Deepfake Detection

This project is a web application for detecting deepfake videos using deep learning techniques. The application is built with Python Flask and includes features like user authentication, video upload, and deepfake detection using a trained model.

## Features

- User Authentication (Login/Signup)
- Video Upload and Processing
- Deepfake Detection using Deep Learning
- User Profile Management
- Contact Form

## Tech Stack

- Python Flask
- SQLite Database
- PyTorch (Deep Learning)
- HTML/CSS/JavaScript
- Custom Trained Model (97% Accuracy)

## Project Structure

```
├── app.py                 # Main Flask application
├── models/               # Deep learning model files
├── static/              # Static assets (CSS, JS, images)
├── templates/           # HTML templates
└── uploaded_videos/     # Storage for uploaded videos
```

## Setup and Installation

1. Clone the repository
2. Install requirements: `pip install -r requirements.txt`
3. Run the application: `python app.py`
